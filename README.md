## 📖 Project Overview

This project is a fully responsive, multi-section website for a landscaping business called **GreenScape Landscaping**. It was designed and developed to simulate a real-world client project, focusing on:

- Clean and modern UI/UX design
- Component-based architecture
- Responsive layouts across all devices
- Professional code structure and organization

The goal of this project is to demonstrate **frontend development skills** using React and CSS while building a realistic, client-ready website.

---

## ✨ Features

### 🎯 Core Features

- Smooth scrolling navigation between sections
- Fully responsive design (mobile, tablet, desktop)
- Modular and reusable React components
- Clean and structured CSS using BEM methodology

### 💡 UI/UX Features

- Hero section with background image and strong call-to-action
- Services section with detailed feature cards
- Projects gallery showcasing completed work
- About section highlighting company experience and credibility
- Testimonials section displaying client feedback
- Contact form with validation and user interaction
- Footer with navigation, contact info, and social links

### 🎨 Design Highlights

- Clean, nature-inspired color palette (greens, neutrals)
- Consistent typography and spacing system
- Smooth hover effects and transitions
- Strong visual hierarchy for readability and flow
